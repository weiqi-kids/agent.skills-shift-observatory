# Mode：career_strategy / 求職策略建議

## Mode 定義表

| 項目 | 說明 |
|------|------|
| **Mode name** | career_strategy / 求職策略建議 |
| **Purpose and audience** | 將技能缺口、補課路徑、可投職缺串連，供求職者和轉職者規劃職涯路徑 |
| **Source layers** | 全部 14 個 Layer（需要最全面的跨來源綜合分析） |
| **Automation ratio** | 40% — 高度依賴 AI 綜合判斷，需要人工審核 |
| **Content risk** | **最高風險 Mode** — 職涯建議直接影響個人決策，錯誤建議可能造成嚴重後果 |
| **Reviewer persona** | 使用者誤導風險審核員 + 法規與責任審核員 + 自動化邊界審核員 |

---

## 風險等級聲明

> **本 Mode 為系統中風險等級最高的報告。** 職涯建議直接影響個人的時間投資、金錢支出、甚至人生方向。AI 系統基於有限數據產出的「建議」存在根本性限制——無法考量個人背景、能力、偏好、家庭狀況等關鍵因素。
>
> **所有輸出內容必須嚴格遵守以下原則：**
> 1. 所有「建議」必須明確標註為「基於數據觀測的參考方向」
> 2. 禁止使用確定性語氣（如「你應該」、「必須」、「一定要」）
> 3. 禁止做出特定時間範圍的預測（如「3年內此職業將消失」）
> 4. 禁止推薦特定付費課程或培訓機構
> 5. 免責聲明必須完整且顯眼

## 自我審核 Checklist

產出報告前，必須逐項確認。**本 Mode 為最高風險，所有項目均為強制性。**

### 語氣與表述審核
- [ ] **無確定性建議**：所有「建議」是否明確標註為「基於數據的參考方向」？是否避免了「你應該」、「必須」、「一定要」等用語？
- [ ] **無確定性預測**：是否有任何看似確定性的職涯預測？（若有，必須改為條件式表述，如「若此趨勢持續」、「基於目前觀測」）
- [ ] **無歧視性內容**：是否有任何可能被解讀為歧視性的建議？（如性別、年齡、學歷歧視）
- [ ] **無恐嚇性語氣**：是否有任何可能引發不必要焦慮的表述？（如「該職業即將消失」）

### 數據與來源審核
- [ ] **來源可追溯**：每個數據點是否可追溯到具體的 Layer 或 Mode？
- [ ] **無幻覺內容**：是否有任何「學習資源」、「轉職路徑」是 AI 編造而非來自觀測數據？
- [ ] **學習資源可驗證**：所有列出的學習資源是否為公開可驗證的平台？是否包含任何付費課程推薦碼？
- [ ] **觀測 vs 因果**：是否混淆了觀測結果與因果關係？（例如：「A 技能需求上升」不等於「學 A 就能找到工作」）

### 結構與完整性審核
- [ ] **免責聲明完整**：免責聲明是否包含所有 7 點？是否位於報告末端且清晰可見？
- [ ] **重要聲明置頂**：報告開頭是否包含重要聲明？
- [ ] **五大向量完整**：是否涵蓋了所有 5 個 AI 取代向量的分析？
- [ ] **不確定性提醒**：每條轉職路徑是否包含「不確定性提醒」？

### 倫理審核
- [ ] **無利益衝突**：是否有任何內容可能被視為為特定機構或平台做宣傳？
- [ ] **弱勢群體考量**：建議是否考慮了不同經濟條件的讀者？（例如：不應假設所有人都有資源學習新技能）
- [ ] **無倖存者偏差**：是否避免了以成功案例推論一般性建議？

## 執行注意事項

1. **依賴前序 Mode**：本 Mode 應在 climate_index、skills_drift、industry_segments、salary_bands 之後執行，以便引用其分析結果。若前序 Mode 未產出，需在報告中說明「本週為簡化版，因前序分析尚未產出」。

2. **學習資源原則**：
   - 僅列出以下類型的資源：官方文件、知名開源專案教程、主流學習平台名稱（如 Coursera、edX、freeCodeCamp）
   - 不列出：特定付費課程、個人部落格教程、未經驗證的新興平台
   - 不評價：各平台的教學品質、性價比

3. **轉職路徑謹慎度**：
   - 僅基於技能重疊度和市場需求數據推論，不做主觀的「適合度」評估
   - 每條路徑必須包含「不確定性提醒」
   - 不推薦具體的轉職時間點

4. **AI 取代向量評估**：
   - 僅基於可觀測的數據（職缺變化、技能需求變化）
   - 不做超過 1 年的預測
   - 避免使用「將被取代」、「即將消失」等絕對用語

5. **多元觀點**：
   - 當數據支持多種解讀時，應呈現多元觀點而非選擇一種
   - 明確標註哪些是數據直接支持的觀察、哪些是推論

6. **禁止行為**：
   - 禁止推薦具體公司或職位
   - 禁止預測特定產業或角色的「消亡時間」
   - 禁止暗示只有一條「正確」的職涯路徑
   - 禁止使用「黃金」、「鐵飯碗」、「夕陽」等帶有強烈主觀判斷的用語
