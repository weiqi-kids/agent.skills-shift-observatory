Rolling Insights | Intermediate–Senior Node.js Developer (Sports Data + Generative AI) | REMOTE (Canada) | Full-time

Rolling Insights builds modern sports data products and APIs used by fantasy platforms, sportsbooks, and media companies. We work with large-scale real-time datasets and are expanding into generative AI-driven collection engines to help scale and automate our data assets.

We’re hiring an intermediate–senior Node.js developer with strong backend experience and comfort working across the stack when needed.

What you’ll do:
- Build and maintain Node.js services and APIs powering live sports data products
- Work with event-driven pipelines, data enrichment, and high-throughput systems
- Collaborate on new AI-assisted features and workflows using LLMs
- Contribute to architecture decisions and ship production features end-to-end

Tech stack:
Node.js, JavaScript, React, Azure, PostgreSQL, real-time data systems, LLM integrations

Ideal background:
- 3+ years professional Node.js experience
- Strong API design and database fundamentals
- Experience with cloud infrastructure (Azure preferred)
- Interest in sports data, analytics, and applied generative AI
