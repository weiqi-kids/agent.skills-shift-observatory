# B009 行銷企劃培育主管(年代健康生活)

## 基本資訊
- **公司/機關**: 年代網際事業股份有限公司
- **地點**: 台北市內湖區
- **薪資**: 35,000 ~ 45,000 元/月
- **類別**: 創意設計
- **發布日期**: 2025-11-15

## 工作內容
工作內容
1.負責行銷企劃案的撰寫、規劃、籌備、執行，並對其效益進行分析與建議
2.進行產業競爭分析及市場調查分析
3.負責相關文宣資料的規劃與製作
4.社群網站操作如FB、IG、YT、LINE等廣告策劃與投放
5.經營與管理品牌、產品及公司整體形象推廣，增加社群會員黏著度

工作技能
1.網路活動規劃與執行
2.品牌知名度推廣
3.社群媒體經營管理
4.網站流量成效追蹤
5.廣告企劃案╱文案撰寫
6.廣告效益評估

其他條件
1.熟悉網路世界，時下熱門話題不放過《Youtuber/Podcast/PTT/Dcard等》
2.能配合團隊工作並了解廣告業的生活型態
3.抗壓力強，能為了琢磨好作品而一改再改
4.具良好的溝通協調能力，願意學習新事物，重視團隊合作精神
5.具醫藥保健市場相關業務經驗者尤佳
6.需出差，累積時間未定
