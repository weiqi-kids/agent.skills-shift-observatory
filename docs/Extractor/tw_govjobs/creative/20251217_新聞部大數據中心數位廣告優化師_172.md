# 新聞部大數據中心數位廣告優化師

## 基本資訊
- **公司/機關**: 中天電視股份有限公司
- **地點**: 台北市內湖區
- **薪資**: 面議
- **類別**: 創意設計
- **發布日期**: 2025-12-17

## 工作內容
1.優化現有聯播網收益(包含廣告位置、價格)
2.與技術團隊、外部廠商合作、評估與導入新的聯播網
3.與編輯、技術團隊合作，增加頁面價值及頁面流量
4.廣告素材管理(需熟悉DFP、Adsense、ADX、GA)與成效優化(含廣告代碼製作)
5.即時監控廣告效益，網站報表分析與網路廣告相關提案建議

其他條件
1具備數位廣告版位操作暨數位廣告投放經驗3年以上
2.熟悉數位廣告運作邏輯
3.熟悉大型媒體聯播網變現
4.對於以數據分析為基礎創造收入有興趣
5.具備專案管理能力(專案時間/進度控管)、協商談判能力、廣告效益評估
6.有廣告投遞經驗者優選
7.具團隊合作、溝通能力
8.具備Google Ads 多媒體廣告認證，GA分析個人認證資格優選

## 資格條件
