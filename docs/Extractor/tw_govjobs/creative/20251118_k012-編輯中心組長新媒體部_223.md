# K012 編輯中心組長(新媒體部)

## 基本資訊
- **公司/機關**: 年代網際事業股份有限公司
- **地點**: 台北市內湖區
- **薪資**: 40,000 ~ 45,000 元/月
- **類別**: 創意設計
- **發布日期**: 2025-11-18

## 工作內容
工作內容
1. 新聞挑選、編輯、撰寫；內容控管：官網及手機APP新聞內審核發佈、直播活動溝通協調，分析流量，強化內容 
2. 規劃各社群媒體的貼文主題，社群媒體執行成效追蹤與數據分析
3. 發佈新聞動態，定期與粉絲互動，策劃並執行相關線上和線下推廣活動，通過有效運營手段提升粉絲活躍度

其他條件
1.有社群操作經驗(粉絲團操作、APP操作、網站首頁維護、廣告行銷)
2.具基本影像處理、影片剪輯能力
3.即時了解社群媒體的動向及新聞時事
4.對於網路話題有敏銳度者
5.活動、企劃、行銷的數據整理和成效分析
