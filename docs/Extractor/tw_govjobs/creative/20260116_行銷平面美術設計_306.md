# 行銷平面美術設計

## 基本資訊
- **公司/機關**: 柏美國際有限公司
- **地點**: 台北市文山區

- **發布日期**: 2026-01-16

## 工作內容
工作內容：
負責公關活動之平面設計及執行活動部分，如：企業活動、產品上市、獎勵活動、主題餐會、開幕剪綵活動、慶典儀式、展示會、慈善贊助、募款或是其它公眾活動等）。
1. 會使用Photoshop、Illustrator等各種多媒體製作工具，熟MS-Office且具單獨處理突發案件之能力、有SENSE的人。 
2. 文宣品、繪圖、主視覺設計、文案，手冊、專刊排版。 
3. 簡易宣傳影片拍攝製作與剪輯。
4. 協助行銷公關活動企劃與執行。
5. 態度積極、認真、有責任感、有禮貌，且能自我控管。
6. 主管交辦事項。
