#  職業安全衛生管理員(台北)

## 基本資訊
- **公司/機關**: 台灣大創百貨股份有限公司
- **地點**: 台北市中正區
- **薪資**: 32,000 ~ 42,000 元/月
- **類別**: 製造生產
- **發布日期**: 2026-01-13

## 工作內容
1.擬訂、規劃、督導及推動安全衛生管理事項並指導有關部門實施。
2.各式安全衛生作業標準制訂、健康檢查之規劃、管理及促進。
3.規劃、執行新進/在職人員教育訓練級證照管理。
4.職業災害、虛驚事故之調查處理及統計分析。
5.主管交辦管理業務事項等。
※具零售賣場經驗者優先錄用

其他條件
1.具乙級職業安全管理證照
2.具零售業.賣場經驗者佳
3.具跨部門溝通協調能力
4.認同台灣大創品牌精神與理念
5.新進人員須於店鋪實習１個月

工作技能
勞工安全相關法規、執行安全衛生督導及稽核、擬定各項安全衛生管理辦法

需出差，一年累積時間未定
