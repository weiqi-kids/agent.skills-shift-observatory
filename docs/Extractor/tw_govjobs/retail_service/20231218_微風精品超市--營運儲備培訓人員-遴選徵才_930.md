# 微風精品超市- 營運儲備培訓人員 遴選徵才

## 基本資訊
- **公司/機關**: 微風廣場實業股份有限公司
- **地點**: 台北市松山區
- **薪資**: 32,000 ~ 36,000 元/月
- **類別**: 零售服務
- **發布日期**: 2023-12-18

## 工作內容
**各項職務說明**
【起司櫃位】
工作內容：
1.進口起司類產品銷售服務
2.起司相關商品進銷貨管理
3.櫃位商品陳列管理
4.喜歡起司對飲食搭配喜歡研究

【酒櫃專區】
工作內容：
1.進口酒類產品銷售服務
2.酒類相關商品進銷貨管理
3.櫃位商品陳列管理
4.喜愛酒類商品及對於酒類商品有研究

【精肉專業區】
工作內容：
1.負責精肉區的陳列管理，與顧客服務及肉品介紹和說明。
2.精緻肉品區營運及各種肉品分切/包裝/賣場維護與管理。
3.具一年以上生鮮肉品分切處理經驗。

【超人氣日式點心專賣櫃】:甘月堂
工作內容：
1.商品陳列上架/補貨
2.簡單點心料理 :日本和果子 / 鯛魚燒 / 大福蕨餅 製做 
3.商品包裝/收銀作業
4.喜歡日式點心製作及學習者歡迎加入

【商品/乾貨陳列】
工作內容：
1. 商品進貨銷售庫存管理。
2. 陳列上架補貨。
3.檔期活動現場販售執行。
3. 現場營業支援
4.料理商品開發規劃。
5.熱愛美食料理或有濃厚興趣。
