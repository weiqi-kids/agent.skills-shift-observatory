# 業務人員 (績效獎金另計)

## 基本資訊
- **公司/機關**: 駿樺旅行社有限公司
- **地點**: 台北市松山區
- **薪資**: 29,500 ~ 32,000 元/月
- **類別**: 零售服務
- **發布日期**: 2026-01-05

## 工作內容
1.開發潛在客戶，拓展市場，以達成業績目標
2.定期拜訪同業客戶，維繫穩定客戶關係
3.業務接洽及訂單處理
4.旅遊產品報價及產品展示，並處理帳款回收相關事宜
5.旅遊業務推展，傳達及說明公司各項業務重大訊息、活動及產品
6.公司不定期舉辦旅展，參與旅展等活動，介紹公司旅遊產品
7.業務人員除本薪，另有額外加上業務獎金制度
