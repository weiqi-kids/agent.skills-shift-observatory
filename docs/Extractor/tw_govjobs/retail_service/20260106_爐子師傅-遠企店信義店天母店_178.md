# 爐子師傅-【遠企店/信義店/天母店】

## 基本資訊
- **公司/機關**: 食昌企業股份有限公司
- **地點**: 台北市不限
- **薪資**: 52,000 ~ 58,000 元/月
- **類別**: 零售服務
- **發布日期**: 2026-01-06

## 工作內容
1.協助廚師測量食材的容量與重量。
2.依照客人的點單，準備不同菜色所需要的食材，擅長炒爐。
3.於出菜時負責菜餚擺盤或調整份量之工作。
4.協助打包外帶的食物。
5.確保負責區域符合食品安全與衛生。
6.各項廚房器具之清潔、保管與維護作業。
7.主管交辦事宜。

【紅豆食府】各分店:
信義店--台北市松壽路9號7F（新光三越信義新天地A9館）近市政府捷運站
遠企店--台北市敦化南路二段203號B2 (遠企購物中心)近六張犁捷運站
天母店--台北市中山北路六段77號8樓（SOGO百貨天母店）近芝山捷運站
