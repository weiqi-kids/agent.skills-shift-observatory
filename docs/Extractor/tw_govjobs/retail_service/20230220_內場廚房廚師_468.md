# 內場廚房廚師

## 基本資訊
- **公司/機關**: 瑪可士西班牙料理有限公司
- **地點**: 台北市大安區
- **薪資**: 30,000 ~ 60,000 元/月
- **類別**: 零售服務
- **發布日期**: 2023-02-20

## 工作內容
1.負責洗、剝、削、切各種食材，以完成烹飪的前置工作。
2.測量食材的容量與重量。
3.依照客人的點單，準備不同菜色所需要的食材。
4.於出菜時負責菜餚擺盤或調整份量之工作。
5.準備於營業時間內的點菜或宴會所需菜餚。
6.確保所負責類別的食品均符合衛生管理的規定。
7.維持與確保冷凍庫及工作區域的衛生及清潔整理標準。
8.保持烹飪材料的適當存量。
9.清理清潔乾淨廚房一切相關事務。
10. 備料，員工餐，點菜．
