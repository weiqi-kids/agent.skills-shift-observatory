# 晶湯匙（SOGO店）外場正職夥伴33K起

## 基本資訊
- **公司/機關**: 共同存在國際餐飲股份有限公司
- **地點**: 台北市大安區
- **薪資**: 33,000 ~ 35,000 元/月
- **類別**: 零售服務
- **發布日期**: 2026-02-04

## 工作內容
台北復興SOGO店：台北市忠孝東路3段300號10樓(SOGO百貨復興店內)
電話:02-8772-5006

【個人特質】
1. 具服務熱忱、細心負責。 
2. 積極正向、活潑開朗。 
3. 有志投入餐飲服務業者。 
4. 願接受公司培訓計畫及升遷制度。 


【工作內容說明】
1. 負責跑單、送餐及聯繫內外場之工作。
2. 負責客人帶位、介紹餐點、點餐、桌邊服務等工作。
3. 於客人用餐完畢後，負責收拾碗盤與清理環境及結賬收銀。
4. 進行簡易餐飲之料理，如：調配飲料、甜點製作等。
5. 提供顧客100%的貼心服務。
6. 接受公司教育訓練及排班。
7. 協助文書處理，如：顧客意見卡Key In。

 
【福利制度】
1. 享勞保、健保、團保、6%勞退基金提撥。
2. 全勤獎金、三節禮券、員工聚餐福利。
3. 工作期間可享員工餐。
4. 員工店內用餐折扣。
5. 績效獎金。
6. 完整的教育訓練，順暢的升遷管道。
