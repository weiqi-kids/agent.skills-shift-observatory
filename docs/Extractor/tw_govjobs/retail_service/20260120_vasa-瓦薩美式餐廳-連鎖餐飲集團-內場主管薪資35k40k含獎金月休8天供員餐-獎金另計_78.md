#  VASA 瓦薩美式餐廳-【連鎖餐飲集團 內場主管】薪資35K~40K(含獎金)月休8天，供員餐 獎金另計

## 基本資訊
- **公司/機關**: 商邦餐飲有限公司
- **地點**: 台北市內湖區
- **薪資**: 35,000 ~ 40,000 元/月
- **類別**: 零售服務
- **發布日期**: 2026-01-20

## 工作內容
【職務內容】
1.成本控管：包括食材成本、營運成本、協助內場物料庫存及設備庫存盤點等。 
2.內場管理：確保分店在符合公司文化及精神下正常運作。 
3.教育訓練：針對內場人員進行不定期教育訓練、確保出餐品質及安全衛生。 
4.危機問題反應及處理。 
5.公司內部及外部的溝通管道，對內的上下溝通能力。
6.需有曾擔任一家連鎖店以上企業的內部主管經驗。
7.內場標準sop流程執行。

【工作內容】
1.熟店舖管理、營運
2.富有觀察力，能理性處理客戶訴求，解決問題
3.具備門市行政管理、財務作業、及成本控管等基礎知識
4.管理及教育員工，擅長溝通協調 
5.具有餐飲管理工作經歷
6.擔任店主管代理人

【具備特質】
1.工作態度認真，且積極負責。
2.喜歡與人群互動！
3.熱衷於團隊活動，能與夥伴們溝通，達到好的分工。
4.喜歡學習，對工作抱持著學海無涯的精神！
5.我們喜歡有責任感、有企圖心的夥伴，歡迎與我們一起學習成長！
6.我們需要你保持親切的笑容服務客人~
