# 康是美_信義區/松山/南港門市人員(獎金上看8個月)

## 基本資訊
- **公司/機關**: 統一生活事業股份有限公司
- **地點**: 台北市信義區
- **薪資**: 32,000 ~ 32,000 元/月
- **類別**: 零售服務
- **發布日期**: 2024-04-25

## 工作內容
※工作內容
1.顧客服務：商品諮詢和推廣、收銀結帳。
2.店務營運：檔期活動執行、商品查檢、補貨與盤點作業。
3.店型維護：店頭陳列佈置、門市環境維護。
4.區組支援：區組門市之間調動支援。

※工作時間：須輪早、晚班，每日8小時，依照勞基法規定一例一休。

※獎金
1.夜班津貼、商品銷售獎金、營運目標達成獎金、年終獎金、個人績效獎金。
2.任職一個完整年度，獎金上看8個月。

※福利
1.年節賀禮：三節、勞動節、生日 禮金或禮品。
2.福儲信託：員工提撥金額公司再加碼，協助您累積另一筆退休金 。
3.其他福利：員工團體保險、子女教育獎助學金、旅遊補助。

※教育訓練：新進員工將安排門市實習及數位課程訓練。

※職涯發展：透過工作學習專業及經營管理，有機會擔任美麗諮詢師、彩妝師，或往管理職晉升。
