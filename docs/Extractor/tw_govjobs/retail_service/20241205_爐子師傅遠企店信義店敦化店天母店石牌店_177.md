# 爐子師傅【遠企店/信義店/敦化店/天母店/石牌店】

## 基本資訊
- **公司/機關**: 食昌企業股份有限公司
- **地點**: 台北市信義區
- **薪資**: 面議
- **類別**: 零售服務
- **發布日期**: 2024-12-05

## 工作內容
1.協助廚師測量食材的容量與重量。
2.依照客人的點單，準備不同菜色所需要的食材，擅長炒爐。
3.於出菜時負責菜餚擺盤或調整份量之工作。
4.協助打包外帶的食物，必要時進行外送的服務。
5.確保負責區域符合食品安全與衛生。
6.各項廚房器具之清潔、保管與維護作業。
7.主管交辦事宜。食材的準備、處理及烹調中/西式菜餚並負責研發新料理。
