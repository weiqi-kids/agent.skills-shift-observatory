# 【涮乃葉】- 大台北 正職人員

## 基本資訊
- **公司/機關**: 雲雀國際股份有限公司
- **地點**: 台北市不限
- **薪資**: 30,000 ~ 33,000 元/月
- **類別**: 零售服務
- **發布日期**: 2022-01-05

## 工作內容
擴大招募 現場營運正職人員								
雲雀國際餐飲集團旗下品牌 【涮乃葉】、【古拉爵】、【洋食芳鄰】分店遍及全省。								
新進同仁安排基礎訓練、專業職能訓練、衛生安全教育訓練。								
【外場】								
1.基層至專業的桌邊服務與接待工作學習起。								
2.負責招呼接待、點餐、送餐及聯繫內外場之工作。								
3.學習協助店內產品掌控、結帳、盤點及清潔規範等相關事宜。								
4.工作區域整理＆清潔。								
【內場】								
1.負責洗、剝、削、切各種食材，以完成烹飪的前置工作。								
2.依照顧客點購的餐點，調理出色香味形器兼備的美味料理。								
3.於出餐時負責料理擺盤並確認擺盤份量＆品質之工作。								
4.工作區域整理＆清潔。								
在職同仁進行持續性的訓練，促進技能與營運管理能力的提升與精進，								
健全之教育訓練體制，暢通之升遷管道，期待每位同仁與公司一同成長。								
月休 9 -10 天，8 H/日，輪班制，排班依店內安排為主。								
歡迎有服務熱忱與喜愛學習挑戰之夥伴加入我們的行列，有餐飲經驗者亦可。
