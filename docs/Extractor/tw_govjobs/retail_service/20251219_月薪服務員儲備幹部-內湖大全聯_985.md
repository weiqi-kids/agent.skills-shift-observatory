## 工作內容
1. 連鎖速食店產品製作。
2. 門市顧客服務，能配合排班。
3. 年平均月休9-10天(依門市排班)。
4. 月薪服務員薪資$ 34,546起

5. 儲備幹部薪資$ 35,603起
    (月薪$ 31,770起 + 區域津貼1,000 + 月在職獎金$ 2,833；月在職獎金每半年進行發放)
6. 年薪13個月，另依公司及單店營運發放獎金。
