# VASA 瓦薩美式餐廳-【連鎖餐飲集團 外場主管】薪資35K~40K，月休8~10天，供員餐 獎金另計

## 基本資訊
- **公司/機關**: 商邦餐飲有限公司
- **地點**: 台北市內湖區
- **薪資**: 35,000 ~ 40,000 元/月
- **類別**: 零售服務
- **發布日期**: 2026-01-20

## 工作內容
【工作內容】
1.熟店舖管理、營運
2.富有觀察力，能理性處理客戶訴求，解決問題
3.具備門市行政管理、財務作業、及成本控管等基礎知識
4.管理及教育員工，擅長溝通協調 
5.具有餐飲管理工作經歷
6.擔任店主管代理人

【職務內容】
1.外場管理：確保分店在符合公司文化及精神下正常運作。 
2.教育訓練：針對新進人員及外場服務人員進行不定期教育訓練、以使工作人員達到最高標準的服務及表現。 
3.危機問題反應及處理。 
4.公司內部及外部的溝通管道：對外的公關處理能力及對內的上下溝通能力。
5.需有曾擔任一家連鎖店以上企業的內部主管經驗。
6.餐廳現場經營營運管理，外場sop服務流程執行。

其他條件
餐飲連鎖加盟門市訓練培訓人材有經驗者為佳
