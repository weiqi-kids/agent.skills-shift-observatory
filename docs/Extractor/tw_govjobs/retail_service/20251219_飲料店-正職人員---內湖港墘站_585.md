# 【飲料店-正職人員】---內湖港墘站

## 基本資訊
- **公司/機關**: 華記企業有限公司
- **地點**: 台北市內湖區
- **薪資**: 34,000 元/月 起
- **類別**: 零售服務
- **發布日期**: 2025-12-19

## 工作內容
★◎※#13
1.	負責櫃台接待、點餐、收銀結帳，並處理外送平台訂單（UberEats／foodpanda）。
2.	依照標準流程製作各類飲品，包含煮茶、搖飲、煮珍珠及各式配料。
3.	根據配方比例與客製化需求調整甜度、冰量，維持飲品品質一致性。
4.	進行每日備料，如煮茶、補配料、補充糖液、鮮奶及其他物料。
5.	維持吧台及店內環境整潔，包含設備清潔、工具洗滌及垃圾處理。
6.	協助物料盤點、補貨與進貨整理，並遵守食品安全規範（先進先出）。
7.	簡易客訴處理，提供即時協助與良好顧客服務體驗。
8.	門市主管交辦事項等。
