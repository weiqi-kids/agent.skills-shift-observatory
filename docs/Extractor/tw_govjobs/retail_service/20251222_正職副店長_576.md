# 正職副店長

## 基本資訊
- **公司/機關**: 花心善奕企業社
- **地點**: 台北市南港區
- **薪資**: 31,000 ~ 50,000 元/月
- **類別**: 零售服務
- **發布日期**: 2025-12-22

## 工作內容
※具有餐飲＂管理＂一年以上經驗者，起薪36000元。
※具有餐飲＂兼職＂半年經驗者，起薪31000元。
※無任何餐飲經驗者，起薪依照勞基法計算。
接受企業培訓並執行各項工作任務，為將來準備要升任的管理職預先準備。
1. 外場：檢查開餐前準備工作，工作站內外清潔環境。
2. 內場：前置備料作業管理，出餐及廚務相關工作，檢查環境乾淨度。
3. 外務：南軟辦公大樓出攤、機車外送餐點、發宣傳物與第一線面對客人銷售介紹
4. 行銷：影音、發文、活動發想。
5. 活動：月度、年度活動討論與執行。
6. 管理：正職與兼職人員工作內容評比管理。
