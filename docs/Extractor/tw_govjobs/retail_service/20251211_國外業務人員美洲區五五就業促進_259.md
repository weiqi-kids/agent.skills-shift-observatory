# 國外業務人員(美洲區)(五五就業促進)

## 基本資訊
- **公司/機關**: 欣技資訊股份有限公司
- **地點**: 台北市大安區
- **薪資**: 面議
- **類別**: 零售服務
- **發布日期**: 2025-12-11

## 工作內容
1. 向指定區域的現有或新合作夥伴/ISV 推廣和銷售 Cipherlab 產品/解決方案。
2. 制定成長策略和計劃。
3. 毫無偏差地遵守所有書面政策和程序，或向上級管理層尋求指導。
4. 在指定的銷售區域開發和管理客戶帳戶/管道，包括制定策略銷售原則，包括漏斗開發、里程碑和結束。
5、千方百計培育新業務！您將進行推銷電話以及致電公司產生的潛在客戶。
6. 廣泛的後續活動，以推進正在進行的銷售機會並推動完成。
7. 確保我們的聯絡人管理資料庫中為潛在客戶和活躍客戶提供及時、有價值、準確和最新的數據。
8.提交每週進度報告並確保數據準確
9. 與銷售、行銷、產品管理和相關職能部門的其他人員建立協作工作關係，以確保成功的客戶體驗。
10.參加銷售會議、訓練研討會和輔導活動。
11. 經常出差到指定地區的客戶現場，並參加研討會和貿易展。
