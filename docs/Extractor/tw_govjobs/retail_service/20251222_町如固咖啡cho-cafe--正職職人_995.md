# 町如固咖啡(Cho Cafe) -正職職人

## 基本資訊
- **公司/機關**: 八海國際餐飲股份有限公司
- **地點**: 台北市萬華區
- **薪資**: 30,000 ~ 35,000 元/月
- **類別**: 零售服務
- **發布日期**: 2025-12-22

## 工作內容
★◎※#13
正職職人須具備：(無經驗者可從餐點工作開始培養)
1.對咖啡的熱情無人能擋，喜愛鑽研各式咖啡的專業
2.具備義式咖啡機與手沖咖啡工作經驗，能接受挑戰並自我要求日益求精
3.想沈溺在被咖啡包圍的環境與認真配合團隊合作讓每一位無論什麼原因來到店裡的客人都被產品或服務感動地離開
＊具備英文能力待優

正職職人職務：
1.吧台工作與設備使用清潔維護，並協助出餐
2.傳承創辦人的初心，用心製作符合水準的餐點與飲料，並能介紹給客人
3.管理咖啡豆的進出入庫，使冠軍大師烘製的產品維持在最佳賞味期
4.參與咖啡豆單的選定與店內咖啡銷售的規劃執行

正職人員為傳遞初心的代言人
1.給予客人愉快的用餐感受，顧客接待服務、電話接聽、送餐等等
2.即時依據顧客點單，完成製作餐點，讓顧客享受我們用心準備的餐點。 
3.與同事一同維護店內環境整潔，讓大家一起享受美美又乾淨的環境。
4.學習製作店內銷售商品(掛耳包、咖啡禮盒、冰滴咖啡、義式咖啡、手沖咖啡等等)。
