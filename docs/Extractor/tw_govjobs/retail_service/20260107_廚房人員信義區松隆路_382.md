# 廚房人員（信義區松隆路）

## 基本資訊
- **公司/機關**: 淞美清潔有限公司
- **地點**: 台北市信義區
- **薪資**: 45,000 元/月 起
- **類別**: 零售服務
- **發布日期**: 2026-01-07

## 工作內容
1. 負責準備並烹飪各類中式菜餚，熟悉爐灶、砧板和蒸籠等廚房設備操作
2. 確保廚房的衛生整潔，依循食品衛生與安全規範進行作業
3. 協助食材的洗剝削切與保存，負責庫存管理及食材的盤點作業
4. 參與新菜品研發及菜單規劃，確保菜餚口味與擺盤的品質
5. 定期維護與清潔廚房設備，確保所有設備功能正常使用
6. 按照廚房標準流程執行工作，協助廚房出菜進度的管理
7. 執行廚餘分類回收與環境保護相關措施，注重節能減排
