# 連鎖超商餐飲設備維修工程師(台北總公司)

## 基本資訊
- **公司/機關**: 台新餐飲設備股份有限公司
- **地點**: 台北市內湖區
- **薪資**: 40,000 元/月 起
- **類別**: 零售服務
- **發布日期**: 2026-01-22

## 工作內容
我們是專業的餐飲設備供應商，服務對象為台灣連鎖便利商店，工作內容以門市餐飲設備的安裝、檢修與維修為主。
我們現正招募設備維修工程師，依排程至各門市進行設備檢查、故障排除與基本維護作業，協助維持門市設備正常運作，確保現場營運不受影響。
給予從零開始的機會，歡迎無相關工作經驗者加入，台新將提供完整的教育訓練，從無到有培養您的專業能力。

 1. 設備安裝與維護
負責連鎖超商（7-ELEVEN、OK、美廉社）等品牌的咖啡機、萃茶機、冰箱等設備之安裝、維修與定期保養，確保門市營運順暢。

2. 專案執行與改善
參與設備升級與更換計畫，協助推動專案落實，持續優化設備效能與使用體驗。

3. 技術支援與問題處理
即時回應客戶需求，進行現場故障排除與技術協助，確保問題能在最短時間內解決。

4. 顧客服務與滿意度維護
透過定期回訪與操作指導，協助客戶提升設備使用效率，建立穩定信任關係。

工作待遇含：
外勤維修津貼2，000元，須完成教育訓練並通過考核，經評估可獨立執行外勤維修作業後核給。
