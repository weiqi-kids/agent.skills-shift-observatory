# Café de LUGANO 外場服務人員/酒吧人員/咖啡師（兼職）

## 基本資訊
- **公司/機關**: 文華精品股份有限公司
- **地點**: 台北市松山區
- **薪資**: 時薪 200 元起
- **類別**: 零售服務
- **發布日期**: 2026-03-19

## 工作內容
1. 具服務熱忱和溝通技能，提供客人良好的用餐體驗
2. 主動積極，確切執行並完成主管交辦事項
3. 提供外場服務：接待客人、餐點介紹與點餐工作
4. 內外場溝通橋梁：餐況聯繫，協助跑菜與送餐
5. 客人用餐完畢後清潔並整理環境
6. 酒吧人員需具備調酒技能與酒水相關知識、經驗
7. 具備基礎英文溝通能力

## 資格條件
- 學歷要求：不拘
- 工作經驗：無限制
- 酒吧職位需具備調酒技能

- 不定期舉辦聚餐、抽獎活動
