# 照顧服務員

## 基本資訊
- **公司/機關**: 私立愛吾愛居家長照機構
- **地點**: 台北市不限
- **薪資**: 210 ~ 210 元/月
- **類別**: 零售服務
- **發布日期**: 2025-12-29

## 工作內容
1.居家照顧：協助沐浴、穿換衣服、餐飲服務、協助進食、服藥、大小便處理、陪同就醫、陪同散步、陪同運動、陪同或代購物品、關節活動、翻身拍背、家務服務等。
2.須具備照顧服務員證照。
3.主要服務對象為老人、長照2.0對象。
4.能獨立完成照服工作，具照服工作經驗者佳。
5.依居家照服員個人居住地就近安排。
