# 默爾-內場廚師

## 基本資訊
- **公司/機關**: 馡曄國際股份有限公司
- **地點**: 台北市
- **薪資**: 42000 元/月起（上限面議）
- **類別**: 零售服務
- **發布日期**: 2026-02-23

## 工作內容
1. 負責洗、剝、削、切各種食材，以完成烹飪的前置工作
2. 測量食材的容量與重量
3. 依照客人的點單，準備不同菜色所需要的食材
4. 於出菜時負責菜餚擺盤或調整份量之工作
5. 製餐流程、符合食品安全衛生與製餐流程
6. 學習團隊合作，達成團隊目標
7. 披薩及義大利麵及其他各式前菜餐點製作
8. 其他主管交辦事項
