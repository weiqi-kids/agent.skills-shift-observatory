# 內場廚務員(02/06中徵)

## 基本資訊
- **公司/機關**: 鼎王國際餐飲股份有限公司
- **地點**: 台北市不限
- **薪資**: 41,100 ~ 50,000 元/月
- **類別**: 零售服務
- **發布日期**: 2026-01-09

## 工作內容
1.負責洗、剝、削、切各種食材，以完成烹飪的前置工作。
2.測量食材的容量與重量。
3.依照客人的點單，準備不同菜色所需要的食材。
4.於出菜時負責菜餚擺盤或調整份量之工作。
5.協助打包外帶的食物。
6.早班-最早10:30，最晚至22:00；晚班-最早16:30，最晚至04:30。(每日工時8小時，排班制，時間為班別的區段時間，分別為最早之上線時間與最晚之下線時間，班別時間以各單店現場公告為準。)
