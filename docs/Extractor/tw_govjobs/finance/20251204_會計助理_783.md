# 會計助理

## 基本資訊
- **公司/機關**: 全球保全股份有限公司
- **地點**: 台北市士林區
- **薪資**: 30,000 ~ 35,000 元/月
- **類別**: 金融財務
- **發布日期**: 2025-12-04

## 工作內容
工作任務
收發、記錄及彙整交易產生之原始憑證，並登錄至會計系統製作傳票。
審核有關各項費用支付之發票、單據及帳務處理。
處理廠商貨款或費用等應付款項帳務。
處理客戶應收款項帳務。
處理其他一般會計帳務。
申報營業稅、營所稅等各項稅務。
負責總分類帳及帳冊記錄及憑證之登載、核算及保管。
協助辦理公司工商登記事宜。
協助辦理銀行現金存、提款、匯款轉帳。
協助執行一般現金收付作業。
核對儲存現金與記錄是否相符合。
協助管理零用金異動及撥補作業。
協助製作出納科目餘額表、收款日報表、利息收入調節表、銀行存款收支日報表、銀行往來授信明細表等。
