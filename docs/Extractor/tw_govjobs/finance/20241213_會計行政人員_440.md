# 會計行政人員

## 基本資訊
- **公司/機關**: 雲河商行
- **地點**: 台北市士林區
- **薪資**: 30,000 元/月 起
- **類別**: 金融財務
- **發布日期**: 2024-12-13

## 工作內容
會計行政
1. 每月產出「損益表」、「資產負債表」、「現金流量表」
2. 管理耗材與處理廠商貨款或費用等應付款項帳務
3. 處理客戶應收款項帳務及合法節稅
4. 協助薪酬計算與核發及所得稅務申報
5. 管理公司人員異動或資料異動，處理人員加、退勞健保與團保
6. 整理各項費用發票、單據及協助合作會計師
7. 配合會計師進行稅務申報作業
8. 處理其他一般會計帳務及主管交辦事項
