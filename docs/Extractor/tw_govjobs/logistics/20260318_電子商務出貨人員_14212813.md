# 電子商務出貨人員

## 基本資訊
- **公司/機關**: 祥昌電子股份有限公司
- **地點**: 台北市中正區
- **薪資**: 30000 ~ 33000 元/月
- **類別**: 物流運輸
- **發布日期**: 2026-03-18

## 工作內容
1. 熟悉各大購物平台及物流（蝦皮、PCHOME、官網、黑貓、新竹）後台出貨系統
2. 處理各平台訂單每日備貨、出貨、退貨、換貨、調貨等
3. 系統出貨、系統操作列印及出貨包裝相關工作
4. 超商、宅配訂單電話提醒及追蹤、系統處理回覆
5. 環境整理及清潔，庫存協助整理及相關行政工作
6. 回覆客人私訊回答訂單商品問題
7. 商品上、下架、變價及庫存管理
8. 行銷活動排程，規劃商品活動設計與發想
