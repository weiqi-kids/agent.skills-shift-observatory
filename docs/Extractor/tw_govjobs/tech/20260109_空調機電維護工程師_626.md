# 空調機電維護工程師

## 基本資訊
- **公司/機關**: 浩勝實業股份有限公司
- **地點**: 台北市北投區
- **薪資**: 60,000 ~ 80,000 元/月
- **類別**: 科技資訊
- **發布日期**: 2026-01-09

## 工作內容
工程師主要工作內容簡介：
1.至客戶端執行實驗室空調與機電設備之維護、保養及相關技術作業。
2.依據作業標準流程，進行設備檢測、功能確認與紀錄整理。
3.需配合公司安排之假日或夜間加班調度。
4.需配合短期外縣市出差任務。
5.依公司制度完成教育訓練、技能考核與專業能力培訓。
6.依主管指示執行各項工作及臨時交辦事項。
7.積極參與公司培訓計畫，持續提升專業技術。
8.具備高度誠信，能遵守作業規範與客戶保密要求。
9.具備高度責任感，能獨立完成交辦任務。
10.具備良好之溝通協調能力，能與團隊及客戶維持良好合作。
11.勇於面對挑戰，願意在專業領域中持續成長。

工作領域：
★醫療檢驗實驗室（Biosafety Lab）
★生技製程空調系統（PIC/S GMP）
★醫療單位與高階無塵實驗室

我們提供：
★完整教育訓練與專業養成（無經驗可培訓）
★明確升遷路徑：助理工程師 → 工程師 → 專案工程師

加分資格（非必備，具證照另有加給）
★室內配線技術士（乙、丙級）
★冷凍空調裝修技術士（乙、丙級）
