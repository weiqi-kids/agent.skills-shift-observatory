# IT 解決方案售前技術顧問（Presales／Solution Owner）-【JD-00000172】

## 基本資訊
- **公司/機關**: 威策電腦股份有限公司
- **地點**: 台北市中山區
- **薪資**: 45,000 ~ 65,000 元/月
- **類別**: 科技資訊
- **發布日期**: 2026-01-16

## 工作內容
如果你喜歡站在業務與技術之間，以專業協助客戶評估方案、影響決策，而不只是進行系統設定，這個角色會很適合你。

【工作內容】
• 與業務團隊密切合作，深入理解企業客戶需求，規劃整體 IT 解決方案架構
（包含雲端、虛擬化、資安、網路、備份等）。
• 參與客戶簡報、方案說明、POC 測試與技術建議書撰寫，以專業觀點影響並推進專案決策。
• 支援專案投標、技術諮詢與解決方案設計，協助客戶導入或雲端搬遷規劃。
• 整合原廠、內部技術與專案資源，與團隊共同完成專案。
• 持續學習新技術，參與原廠培訓與認證，精進售前顧問與解決方案能力。

【工作地點與型態】
• 上班地點：台北市中山區（依業務行程彈性安排）
• 本職務為專案型工作型態，無需每日進公司
• 依專案需求進行客戶拜訪或內部會議（雙北／桃園／新竹為主）

【我們希望你具備】
✔ 基本條件
• 大學以上學歷，資訊相關科系尤佳。
• 具 Presales 經驗，或具 IT 技術背景並有志發展為售前顧問者
• 熟悉 Windows／Linux 伺服器、網路與基本資安架構概念。
• 具備良好溝通與簡報能力，能與業務及客戶進行技術層面討論。
•本職務角色重心在 售前顧問與方案架構規劃，不以長期維運或單純技術支援為主要職責。

✔ 加分條件
• 曾參與 Solution Sales、專案型售前或標案支援。
• 具 POC 規劃、技術文件或建議書撰寫經驗。
• 具相關原廠或技術認證者尤佳。

【薪資與福利】
• 月薪 45，000～65，000 元（依資歷與能力核定）
• 年度營運績效獎金、專案獎金、原廠認證補助。
• 完善培訓與技術交流資源，快速累積專業實力。

【為什麼加入我們｜威策電腦】
• 產業定位明確：專注企業 IT 整合、雲端與資安，專案類型多元
• 專業成長導向：重視技術深度與顧問能力養成
• 團隊合作文化：Presales 與 Sales 並肩作戰，不是單打獨鬥
