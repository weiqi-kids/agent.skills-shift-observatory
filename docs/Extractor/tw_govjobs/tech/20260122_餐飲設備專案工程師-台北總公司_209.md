# 餐飲設備專案工程師 (台北總公司)

## 基本資訊
- **公司/機關**: 台新餐飲設備股份有限公司
- **地點**: 台北市內湖區
- **薪資**: 40,000 元/月 起
- **類別**: 科技資訊
- **發布日期**: 2026-01-22

## 工作內容
台新餐飲設備為專業餐飲設備供應商，提供餐廳後場整體規劃、設備銷售與施工安裝服務。您將負責專案現場的協調、安裝監督及客戶溝通，確保餐飲設備工程順利完成。

工作內容：
1. 餐廳廚房後場工程之現場會勘、丈量與施工規劃，核對設計圖面與現場條件，並配合水電施工單位進行放樣作業。
2. 餐飲設備（中西式爐具、不鏽鋼檯面、冷凍冷藏設備、排煙系統等）之進場協調、安裝、水平調整及水、電、瓦斯二次端連接。
3. 審核專案工程相關水、電、瓦斯配置數據，並進行設備位置、高度與數量之覆核，確保施工品質與使用安全。
4. 與設計單位、業主、工班及協力廠商溝通協調，召開工程協調會議，控管工程進度與品質，並即時回報專案狀況。
5. 執行設備試車測試，協助工程驗收、文件整理與專案交付，並對客戶進行設備操作與基本教育訓練。
6. 支援業務團隊進行現場技術評估，提供專業建議，協助專案前期規劃與執行。
