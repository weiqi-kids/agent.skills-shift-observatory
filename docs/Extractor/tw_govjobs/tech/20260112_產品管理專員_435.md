# 產品管理專員

## 基本資訊
- **公司/機關**: 臺灣造績股份有限公司
- **地點**: 台北市大安區
- **薪資**: 35,000 ~ 45,000 元/月
- **類別**: 科技資訊
- **發布日期**: 2026-01-12

## 工作內容
1. 使用專案管理工具(JIRA、GIT、confluence、Trello等)管理資源與時程
2. 確認規格，並產出規格書 (SRS、User story)
3. 跟設計師、工程師合作，提供輔助文件，包含wireframe、workflow…等
4. 協助規劃專案工作流程及時程
5. 進行需求變更管理及風險控制

需電子電機系、資訊工程系、應用數學系、通訊工程系、其他工程相關學系

【條件】
1. 具文書處理能力，撰寫規格書、系統操作手冊
2. 工作經驗三年以上，需含一年以上產品管理或專案管理相關工作經驗
3. 思維活躍、積極主動、良好的團隊協作和抗壓性
4. 良好的邏輯思維、條理性和溝通能力
5. 曾參與敏捷式開發流程(Agile，Scrum) ，具設定及執行 product backlog經驗
6. 從使用者角度思考產品功能
7. 了解軟體開發生命週期 (SDLC)
8. 具備 UI/UX 的基本知識與觀念
9. 有責任感與紀律，自我要求高
 
【加分條件】
1. 有跨國、跨團隊合作的經驗
2. 具備英文溝通能力
3. 資訊相關科系畢、具開發經驗及技術背景尤佳
4. 有Scrum相關證照
5. 有JIRA/Confluence等專案管理工具使用經驗
6. 有網站或平台產品經驗

## 資格條件
