# 資訊系統人員

## 基本資訊
- **公司/機關**: 阿米巴資訊股份有限公司
- **地點**: 台北市松山區
- **薪資**: 29,500 ~ 29,500 元/月
- **類別**: 科技資訊
- **發布日期**: 2025-12-16

## 工作內容
1.系統測試、資料建置及軟硬體維護等資訊相關工作
2.資料庫維護、出版、修改、校正。
3.客戶關係維護執行 (連絡、資料處理寄送、問題解決及其它相關行政事務)
4.執行其他主管交辦事務

職務需求
1.具網頁美編技術能力者優先錄取。懂SQL 及Linux 語法、PostgreSQL佳。
2.資料關聯邏輯強、細心、擅長分析及除錯
3.熟悉PDF 編輯(Adobe Acrobat) 
4.懂SQL語法、pgAdmin PostgreSQL Tools 操作、Linux 作業系統基本語法

其他條件: 
1.須工作態度良好、細心且配合度高
2.個性主動積極獨立、謹慎、細心並具高度學習意願 
3.能有效分配、運用自己的時間及資源，且有效率的完成主管交辦工作 
4.能清楚表達自己的想法並使他人了解，並正確解讀他人所傳達的資訊
