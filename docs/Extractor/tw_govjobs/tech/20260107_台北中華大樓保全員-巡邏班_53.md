# 台北中華大樓保全員( 巡邏班 )

## 基本資訊
- **公司/機關**: 駿衛保全股份有限公司
- **地點**: 台北市萬華區
- **薪資**: 36,000 ~ 38,000 元/月
- **類別**: 科技資訊
- **發布日期**: 2026-01-07

## 工作內容
1. 負責大樓內外巡邏，確保建築物及周邊環境的安全，包含門窗檢查與人員進出監控。
2. 監控安全設備操作，如監控器與門禁系統，及即時回應異常情況。
3. 熟悉安全與消防程序，能進行緊急事件處理，如防止入侵、竊盜與火災狀況。
4. 負責安全事件紀錄的整理與報告，並遵守相關安全政策與程序。
5. 協助訪客服務，包含訪客登記、指引與住戶問題協調。
6. 執行建築設備的日常監督及提供大樓綜合管理服務。
7. 需具基本電腦操作能力，能運用系統完成日常任務與紀錄。
