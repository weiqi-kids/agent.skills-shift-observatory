# 通訊工程師

## 基本資訊
- **公司/機關**: 正通科技股份有限公司
- **地點**: 台北市大同區
- **薪資**: 32,000 ~ 45,000 元/月
- **類別**: 科技資訊
- **發布日期**: 2025-09-09

## 工作內容
(1) 參與及執行電信系統（含4G/5G行動系統、無線電、及弱電相關系統）增波器產品訊號涵蓋及工程規劃、現場實地查勘、設備設定、產品查修、網管監控，各類型室內涵蓋為主。  
(2) 執行無線網路量測、Log資料分析、優化調整。 
(3) 配合電信系統建設場域執行工程專案管理，參與電信系統設計、工程現場溝通協調、及專(標)案的文件編撰與管理。從期初規畫、建設、驗收、至維運保固。 
(4) 參與進行客戶問題、設備技術問題、及工程建設問題之分析與解決。 
(5) 執行電信產品測試，含產品品質測試。產品與系統的整合測試。支援射頻產品型式認證。
