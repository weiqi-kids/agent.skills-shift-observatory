# 社區警衛(55plus職場支持輔導計畫)

## 基本資訊
- **公司/機關**: 延平花園大廈管理委員會
- **地點**: 台北市中正區
- **薪資**: 29,500 元/月 起
- **類別**: 科技資訊
- **發布日期**: 2025-12-11

## 工作內容
1.	安全巡邏
依排班計畫，定時巡邏社區範圍，確保社區內所有公共設施、安全設施運行正常，並報告任何異常情況。
2.	監控設備監視
確保設備運行正常，並及時發現並處理可疑人員或事件。
3.	維持社區秩序
負責社區內的秩序維護，阻止非法活動並協助處理社區內的衝突或糾紛，必要時報警處理。
4.	處理緊急情況
需迅速反應處理緊急情況，包括火災、自然災害、設備故障等，並協助居民撤離及聯繫相關應急部門。
5.	門禁管理
需負責管理社區的門禁系統，確保進出人員及車輛的合法性，並協助來訪的住戶及訪客辦理登記。
6.	居民服務/郵件收發
需提供居民必要的協助，如協助遺失物品查找、協助住戶聯繫緊急服務等。
7.	報告與紀錄
需每日完成巡邏報告及其他工作紀錄。
8.	協助其他社區活動
需要時協助社區委員會的其他活動，如社區集會、安全宣導等。
