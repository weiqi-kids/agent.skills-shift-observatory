- **薪資**: 面議

## 工作內容
【技術顧問，運用技術達成專案開發】
掌握並運用流行前後端技術、資料庫操作進行0到1專案開發
具備三年以上實務經驗累積佳
前端 - JavaScript / HTML / CSS (有Vue/React/Angular框架經驗者佳)
後端 - .Net MVC / .Net Core / .Net 6以上版本
資料庫 - MSSQL / MySQL / Oracle 其中一種關聯式資料庫運用

【從ME到WE，攜手團隊協作】
根據SA分析後的需求文件及SD提供之架構與共用模組進行全端網頁開發
搭配前後端分離架構、版控及單元測試等工作流程
主動確認需求方向並產出符合客戶需求及效能要求的專案程式碼

【透明溝通，勇於求助】
遇到問題會自我整理筆記並勇敢提問
可說明嘗試方向、預期成果及當前困難
清晰理解專案內容並透過圖像或文字整理對焦
除了接收需求外，能提出對於需求不合理或是更好的修改建議進行討論

【問題解決】
在面對程式問題時，展現獨立處理能力
透過網路搜尋、AI工具及專家討論，迅速找尋最佳解決方案，成為團隊的技術支柱
