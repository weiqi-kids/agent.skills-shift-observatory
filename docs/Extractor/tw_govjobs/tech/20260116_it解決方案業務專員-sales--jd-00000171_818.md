# IT解決方案業務專員( Sales )-【JD-00000171】

## 基本資訊
- **公司/機關**: 威策電腦股份有限公司
- **地點**: 台北市中山區
- **薪資**: 40,000 ~ 60,000 元/月
- **類別**: 科技資訊
- **發布日期**: 2026-01-16

## 工作內容
【關於這個角色】
✔ 聚焦企業 IT 解決方案與顧問式銷售模式
✔ 與 Presales／技術團隊協作，共同推進專案
✔ 具相關經驗業務，也歡迎無經驗的人才

我們正在尋找希望從「銷售產品」走向「解決客戶問題」，對企業 IT 解決方案有興趣，並願意與技術團隊合作的業務夥伴。

【工作內容】
• 進行企業客戶開發與經營，包含需求訪談與關係維繫
• 與 Presales／技術團隊合作，規劃解決方案並參與專案提案
• 準備報價、簡報資料，並參與客戶簡報與合約洽談
• 協調內部資源，追蹤專案進度並確保專案順利推進
• 維繫既有客戶，發展後續合作與專案機會
• 此角色將逐步培養成能獨立經營客戶、整合內外部資源的 Solution Sales。

【工作地點與型態】
• 上班地點：台北市中山區（依業務行程彈性安排）
• 本職務為專案型工作型態，無需每日進公司
• 依專案需求進行客戶拜訪或內部會議（雙北／桃園／新竹為主）

【我們在找這樣的你】
✔ 基本條件
• 專科以上學歷（資訊、資管、商管、行銷相關科系尤佳）
• 對企業 IT 服務、解決方案或專案型銷售有興趣
• 具良好溝通能力，能與客戶及內部團隊協調合作
• 具備基本英文閱讀能力，可查閱技術或產品文件

✔ 加分條件
• 具 B2B / B2C 業務、Solution Sales 或專案型銷售經驗
• 曾接觸雲端、資安、系統整合或資訊服務產業
• 具簡報提案、報價或客戶經營經驗

【培訓與發展機制】
• 依個人背景與經驗，規劃對應的培訓與任務安排
• 前期由資深業務帶案熟悉產品、解決方案與銷售流程
• 隨能力成長，逐步從支援角色轉為專案主責或客戶經營角色
• 提供明確的業績目標、發展路徑與對應獎金機制

【薪資與獎金】
• 月薪 40，000～60，000 元（依資歷與能力核定）
• 績效獎金

【為什麼加入我們｜威策電腦】
• 產業領先：專注企業 IT 整合、雲端與資安，與多家國際大廠合作
• 成長明確：完善培訓、透明升遷制度，讓努力有方向
• 團隊合作：重視跨部門協作，不是單打獨鬥的業務文化
• 專案多元：可接觸不同產業客戶，累積實戰經驗
