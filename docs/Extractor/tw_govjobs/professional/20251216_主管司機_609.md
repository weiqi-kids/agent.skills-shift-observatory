# 主管司機

## 基本資訊
- **公司/機關**: 世群貿易股份有限公司
- **地點**: 台北市中正區
- **薪資**: 42,000 ~ 42,000 元/月
- **類別**: 專業服務
- **發布日期**: 2025-12-16

## 工作內容
1. 擔任主管司機，負責日常主管及家人的行程接送。（可以配合偶爾例假日及晚上聚會的用車需求 尤佳）
2. 需配合執行主管交辦的事務，於執行事務時所獲悉公司的各項相關資料及私人的訊息，均需謹守保密原則。
3. 聰明負責、個性開朗、工作積極、品行端正。
4. 應對有禮儀、具敬業精神及工作倫理且無不良嗜好。
5. 時間觀念佳，熟悉台北市、新北市的交通路線。
6. 負責駕駛車輛的清潔保養及維修事宜，確保車況良好。
7.行車以安全舒適為最高原則。
8.有開過房車的經驗。
9.未出外勤時，協助處理公司內部庶務相關工作。
(請備註 希望待遇)
