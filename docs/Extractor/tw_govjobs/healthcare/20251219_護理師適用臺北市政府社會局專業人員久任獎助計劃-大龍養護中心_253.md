# 護理師(適用臺北市政府社會局專業人員久任獎助計劃)-大龍養護中心

## 基本資訊
- **公司/機關**: 財團法人伊甸社會福利基金會
- **地點**: 台北市大同區
- **薪資**: 42,000 ~ 61,520 元/月
- **類別**: 醫療照護
- **發布日期**: 2025-12-19

## 工作內容
【本職缺申請臺北巿政府社會局身心障礙日間及住宿式照顧服務設施專業人員久任獎助計劃，並依計劃規定工作連續年資，按年加給1萬5，600元至6萬2，400元之個人獎助】

1.住民例行健康照護工作:給藥、傷口換藥、測量生命徵象及記錄、必要時的管路(鼻胃管、尿管)放置及護理照護措施、抽痰無菌技術、抽血。
2.緊急(意外傷害)應變處理:基本急救處理和概念。
3.住民健康評估管理、藥物管理及相關紀錄撰寫。
4.護理相關資料歸檔、整理、分析:基本電腦文書處理及建檔。
5.夜間照顧工作督導:督促生服人員照護上的執行工作。
6.感染管制業務:基本感控概念及措施，大夜生服人員感控稽核。
7.其他主管交辦事項。
※42000為白班薪，另有小夜津貼500元/日、大夜津貼1000元/日，N1-N4每一級另加1000元。※

上班時段:
白    班：0800-1700(休息一小時)；週六、日與國定假日：白班0800-2000
小    夜：1330-2200(休息半小時)。  
大夜班：2300-0800(休息一小時)；週六、日與國定假日：大夜班2100-0800

科系要求:國內外大專(院)校醫護相關科系畢業或以上學歷

其他條件:
1.具護理師證書/執照。  
2.護理相關工作經驗一年以上。
