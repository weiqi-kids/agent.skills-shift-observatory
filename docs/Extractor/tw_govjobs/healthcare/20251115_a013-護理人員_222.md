# A013 護理人員

## 基本資訊
- **公司/機關**: 年代網際事業股份有限公司
- **地點**: 台北市內湖區
- **薪資**: 面議
- **類別**: 醫療照護
- **發布日期**: 2025-11-15

## 工作內容
工作內容
1.勞工之健康教育、健康促進與衛生指導之策劃及實施
2.職業傷病及一般傷病之防治、健康諮詢與急救及緊急處置
3.勞工之預防接種及保健
4.協助雇主選配勞工從事適當之工作 
5.勞工體格、健康檢查紀錄之分析、評估、管理與保存及健康管理
6.職業衛生之研究報告及傷害、疾病紀錄之保存 
7.協助雇主與勞工安全衛生人員實施職業病預防及工作環境之改善  
8.辨識與評估工作場所環境及作業之危害 
9.提出作業環境安全衛生設施改善規劃之建議 
10.調查勞工健康情形與作業之關連性，並對健康高風險勞工進行健康風 險評估，採取必要之預防及健康促進措施
11.協助提供復工勞工之職能評估、職務再設計或調整之諮詢及建議
12.其他上級主管臨時交辦或指定事項

其他條件
1.需具備護理師或護士證照(必備) 
2.具備勞工安全衛生管理師執照者優先錄用 
3.曾接受勞工體格及健康檢查指定醫療機構職業衛生護理研習或事業單位職業衛生護理研習訓練合格者優先錄用 
4.曾接受中央主管機關委託台大護理人員研修中心辦理職業衛生護理組課程63小時者，優先錄用
5.有廠護經驗或行政工作經驗者尤佳
