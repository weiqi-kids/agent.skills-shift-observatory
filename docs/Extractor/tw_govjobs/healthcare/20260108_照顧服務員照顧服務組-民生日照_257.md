# 照顧服務員(照顧服務組)-民生日照

## 基本資訊
- **公司/機關**: 財團法人伊甸社會福利基金會
- **地點**: 台北市松山區
- **薪資**: 34,320 ~ 36,400 元/月
- **類別**: 醫療照護
- **發布日期**: 2026-01-08

## 工作內容
一、工作內容
1.服務對象為18歲以上、失能等級在2到8級之間的身障朋友。
2.針對每位服務對象擬定及執行個別化服務計畫，確保每個人都能獲得適合的服務。
3.設計並帶領團體活動，促進身障朋友的社交互動和身心發展。
4.協助日間服務對象進行生活自理訓練，並提供必要的衛生、身體照顧及健康促進服務。
5.提供社會技能訓練、文康休閒活動，並組織各種節慶活動，讓服務對象有更多機會參與社會活動。
6.執行各項相關的服務配搭工作，確保服務流程順暢。
7.協助環境清潔，保持中心的整潔與安全。
8.完成主管指派的其他工作。

二、其他條件:
1.「照顧服務員專業訓練課程」結業證書或身心障礙教保員訓練結業證書
2.高中（職）以上學校護理、照顧相關科、系、組、所、學位學程畢業。
3.長照服務人員證明
4.基本電腦作業能力
5.到職後須完成「身心障礙支持服務20小時核心課程」
6.具有照顧服務員單一(丙)級技術士者(單一證照加給)
