# B010 生技保健食品業務 /藥局/診所通路(年代健康生活)

## 基本資訊
- **公司/機關**: 年代網際事業股份有限公司
- **地點**: 台北市內湖區
- **薪資**: 35,000 ~ 35,000 元/月
- **類別**: 醫療照護
- **發布日期**: 2025-11-15

## 工作內容
工作內容
1. 拓展市場通路與開發潛在客戶(主要通路為診所及藥局)，以達成業績目標 
2. 定期拜訪經銷客戶，客戶關係管理，對保健食品有興趣者佳！
3. 負責業務接洽及訂單處理、產品報價及產品展示 
4. 負責業務推展，以及現有通路各項業務重大訊息聯繫與行銷服務 
5. 市場情報蒐集與競品、通路布局分析 
6. 支援協助參與相關行銷活動（如：展覽、會議） 

其他條件
1.具3年以上業務銷售相關經驗 
2.熟悉藥品、保健食品銷售通路者優先錄取 
3.具備行銷廣宣概念、助攻客戶銷售能力

工作技能
1.業績目標達成
2.業務或通路開發
3.業績目標分配與績效達成
4.客戶情報蒐集
5.產品介紹及解說銷售
