# 護理師-興隆照顧中心

## 基本資訊
- **公司/機關**: 財團法人伊甸社會福利基金會
- **地點**: 台北市文山區
- **薪資**: 41,000 元/月 起
- **類別**: 醫療照護
- **發布日期**: 2025-12-11

## 工作內容
1.執行例行性護理照護:執行基礎護理照護（量測生命徵象、給藥技術）、執行管路照護、執行傷口/造口照護、每日確實執行預防性照護（抽痰）
2.執行護理評估與處遇:針對現存、潛在之健康問題與健康檢查異常數值、每日確實執行預防性照護（抽痰）
3.外部醫療資源網路建立:連結營養師，協助進行服務對象營養需求之評估、設計、監控與諮詢；連結藥師，協助進行服務對象藥事照護相關業務；連結醫師，協助進行服務對象之診察、開給方劑、治療等相關業務；連結牙醫師，協助進行服務對象口腔保健相關業務；連結居家護理師，協助進行服務對象更管相關業務
4.內部醫療衛生相關事工規劃、督導與培力: 機構內部感控作業與流程、檢核生服員之作業流程 
5.專科以上學校或符合教育部採認規定之國外專科以上學校護理、護理助產、助產科、系畢業，經護理人員考試及格，並領有中央衛生主管機關核發之護理師證書或護士證書。具實務經驗二年以上為佳。
※本職缺申請臺北市政府社會局身心障礙日間及住宿式照顧服務設施專業人員久任獎助計畫，並依計畫規定工作連續年資，按年加給1萬5，600元至6萬2，400元之個人獎助※
