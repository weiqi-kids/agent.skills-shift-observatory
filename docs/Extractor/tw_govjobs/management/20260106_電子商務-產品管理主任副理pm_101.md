# 電子商務 產品管理主任/副理(PM)

## 基本資訊
- **公司/機關**: 山淬股份有限公司
- **地點**: 台北市中正區
- **薪資**: 40,000 ~ 50,000 元/月
- **類別**: 經營管理
- **發布日期**: 2026-01-06

## 工作內容
1. 負責產品線業績達成
2. 產品線 產品管理 及 進/銷/存 管理
   a) 進貨管理：所負責產品線之品牌及商品管理、進貨管理(數量/成本)
   b) 銷貨管理：
        ．商品上下架管理、網站平台營運、賣場文案編輯、活動素材規劃....等
        ．價格制定管理
        ．促銷活動規劃提報、平台毛利商談，例如：自營平台活動規劃(FB/商城/官網) 
            以及與購物中心(pchome/yahoo/momo)
        ．熱銷商品販售方案提報
      c)存貨管理：庫存管理、效期管理、販促案提升迴轉速度
3. 定期成效分析報告 
4. 策略合作洽談：品牌原廠及供應商資源協商整合、通路合作、KOL合作...等
5. 新品牌開發引進
