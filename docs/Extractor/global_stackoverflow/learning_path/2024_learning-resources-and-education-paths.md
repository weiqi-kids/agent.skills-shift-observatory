## 調查面向

## 學習管道分類

- **正規教育**：大學、技術學院的資訊相關科系

- **技術文件**：官方文件、教學指南
- **實作練習**：個人專案、開源貢獻
- **社群資源**：Stack Overflow、技術論壇、部落格
- **AI 輔助學習**：ChatGPT、AI 程式碼助手
