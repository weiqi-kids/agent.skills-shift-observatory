## 競品詳細分析

### 競品 A：LinkedIn Economic Graph

#### 基本資訊

| 項目 | 數據 |
|------|------|
| URL | https://economicgraph.linkedin.com |
| 定位 | 全球經濟的數位呈現，聚焦勞動市場研究與政策建議 |
| 目標受眾 | 政策制定者、政府機構、企業 HR、勞動經濟學研究者 |

#### 數據規模

| 指標 | 數據 |
|------|------|
| 會員數 | 12 億 |
| 企業數 | 6,900 萬 |
| 學校數 | 14 萬 |
| 技能標籤 | 4.1 萬 |

#### 內容策略

| 項目 | 觀察 |
|------|------|
| 主要內容類型 | 政策研究報告、月度就業數據、商業趨勢分析 |
| 更新頻率 | 月度報告 + 不定期研究 |
| 特色內容 | 綠色技能報告、AI 技能趨勢、勞動力報告 |

#### 優勢與特色

1. **數據規模無可匹敵**：12 億會員的實際行為數據
2. **政策影響力**：與政府機構合作，影響政策制定
3. **全球視角**：跨國比較能力強

#### 可改善之處

1. 面向一般求職者的內容較少
2. 數據未完全公開，需付費或合作取得

### 競品 B：Robert Half Salary Guide

#### 基本資訊

| 項目 | 數據 |
|------|------|
| URL | https://www.roberthalf.com/us/en/insights/salary-hiring-trends |
| 定位 | 專業人才市場的權威薪資數據來源 |
| 目標受眾 | 雙重受眾：雇主（薪資競爭力分析）+ 求職者（薪資談判依據） |

#### 內容策略

| 項目 | 觀察 |
|------|------|
| 主要內容類型 | 薪資指南、市場趨勢報告、職涯指南、互動工具 |
| 更新頻率 | 年度薪資指南 + 月度就業報告 |
| 特色內容 | 7 大專業領域薪資區間、Gen Z 職場動機分析 |

#### 優勢與特色

1. **薪資數據權威**：基於實際職位媒合數據
2. **雙重受眾設計**：同時服務雇主和求職者
3. **互動工具**：地區薪資查詢篩選器
4. **職涯內容豐富**：不只數據，還有職涯發展建議

#### 可改善之處

1. 偏重美國市場
2. 以付費服務為導向（數據引導至招聘服務）

### 競品 C：Yale Budget Lab (AI 勞動市場研究)

#### 基本資訊

| 項目 | 數據 |
|------|------|
| URL | https://budgetlab.yale.edu/research |
| 定位 | 學術研究機構，專注 AI 對勞動市場的實證分析 |
| 目標受眾 | 學術研究者、政策制定者 |

#### 內容策略

| 項目 | 觀察 |
|------|------|
| 主要內容類型 | 深度研究報告、政策評估、現況綜述 |
| 更新頻率 | 不定期（學術研究週期） |
| 特色內容 | AI 就業影響月度追蹤 |

#### 優勢與特色

1. **學術嚴謹性**：實證導向，方法論透明
2. **政策影響力**：研究結果影響政策討論
3. **免費公開**：研究成果公開可讀

#### 可改善之處

1. 更新頻率低（非即時）
2. 學術語言，一般讀者不易理解
3. 無行動建議

### 競品 D：BLS (美國勞工統計局)

#### 基本資訊

| 項目 | 數據 |
|------|------|
| URL | https://www.bls.gov |
| 定位 | 美國政府官方勞動統計數據來源 |
| 目標受眾 | 研究者、政策制定者、媒體、企業 |

#### 優勢與特色

1. **數據權威性最高**：政府官方來源
2. **涵蓋範圍廣**：就業、薪資、物價、生產力
3. **歷史數據完整**：長期趨勢分析
4. **免費公開**：所有數據公開可用

#### 可改善之處

1. 僅限美國市場
2. 原始數據，未提供分析解讀
3. 無針對求職者的行動建議

## 關鍵發現

### 1. 差距（他們有我們沒有）

| 項目 | 競品 | 影響 | 建議 |
|------|------|------|------|
| 互動工具 | Robert Half | 用戶無法自助查詢個人化數據 | 考慮加入技能評估工具 |
| 數據規模 | LinkedIn | 數據說服力較弱 | 強調方法論透明、開源優勢 |
| 明確行動建議 | Robert Half | 讀者不知如何行動 | **P0：加入「本週行動清單」區塊** |
| 職涯內容 | Robert Half | 缺乏軟性內容 | 考慮加入面試技巧、履歷建議 |

### 2. 優勢（我們有他們沒有）

| 項目 | 說明 | 如何強化 |
|------|------|----------|
| 每週自動更新 | 更新頻率遠超所有競品 | 首頁強調「每週更新」 |
| 完全開源 | 方法論 100% 透明，GitHub 公開 | 強調「可驗證」的信任感 |
| AI 取代向量框架 | 獨特的 5 維分析框架 | 作為核心差異化賣點 |
| 台灣市場聚焦 | 競品幾乎無台灣數據 | 強化台灣數據源（需修復 104 API） |
| 免費無廣告 | 無商業利益導向 | 建立社群信任 |

### 3. 機會（可以做得更好）

| 項目 | 說明 | 參考對象 |
|------|------|----------|
| 雙重受眾設計 | 同時服務求職者和雇主 | Robert Half |
| 視覺化呈現 | 更多圖表、更少文字 | LinkedIn |
| 互動工具 | 個人化技能評估 | 未來方向 |

## 差異化建議

基於競品分析，建議我們的差異化方向：

1. **「唯一每週更新的開源就業觀測站」**
   - 強調更新頻率 + 開源透明，這是所有競品都沒有的組合

2. **「AI 時代的職涯風險雷達」**
   - 5 維 AI 取代向量是獨特資產，應作為核心品牌識別

3. **「台灣市場最完整的免費就業數據」**
   - 修復台灣數據源後，可成為台灣市場的獨佔優勢
