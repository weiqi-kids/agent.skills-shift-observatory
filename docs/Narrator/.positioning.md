# 品牌定位文件

> 產出日期：2026-02-19
> 階段：0-positioning (Writer)

## 1. 核心資訊

| 項目 | 內容 |
|------|------|
| **網站名稱** | Skills Shift Observatory（技能需求變化觀測站） |
| **一句話描述** | 追蹤全球就業市場變化與 AI 對各產業職缺衝擊的自動化數據分析平台 |
| **核心價值主張** | 讓每個人都能用數據做出更好的職涯決策——透過每週自動分析 15+ 個資料來源，提供客觀、透明、可追溯的就業市場洞察 |

## 2. 目標受眾

### 主要受眾 (Primary)：求職者與轉職者

| 項目 | 描述 |
|------|------|
| **是誰** | 正在找工作或考慮轉職的專業人士，年齡 25-45 歲，具備基本數據閱讀能力 |
| **目標/需求** | 了解市場現況、評估自身技能競爭力、設定合理薪資期望、規劃職涯方向 |
| **痛點** | 1) 資訊分散難以整合 2) 難以判斷市場趨勢真偽 3) 不確定 AI 對自身職業的影響 4) 缺乏客觀數據支持決策 |
| **使用場景** | 每週查看最新報告、準備面試前了解市場行情、評估轉職時機、學習新技能前評估投資報酬率 |

### 次要受眾 (Secondary)：在職專業人士

| 項目 | 描述 |
|------|------|
| **是誰** | 目前有穩定工作但關注市場動態的專業人士 |
| **目標/需求** | 持續了解產業趨勢、評估技能更新需求、掌握 AI 對職業的潛在影響 |
| **痛點** | 工作繁忙無暇追蹤市場、擔心技能過時、不確定是否該主動求變 |
| **使用場景** | 定期（月/季）瀏覽趨勢報告、年度績效評估前參考市場數據 |

### 次要受眾 (Secondary)：HR/企業主管

| 項目 | 描述 |
|------|------|
| **是誰** | 負責招募或人才策略的企業決策者 |
| **目標/需求** | 了解人才市場競爭態勢、制定有競爭力的薪資方案、預測技能需求變化 |
| **痛點** | 薪資調查報告昂貴、數據更新不夠即時、難以預測未來技能需求 |
| **使用場景** | 制定招募策略、薪資調整決策、人才發展規劃 |

### 次要受眾 (Secondary)：研究者/政策制定者

| 項目 | 描述 |
|------|------|
| **是誰** | 學術研究者、政府勞動部門、智庫分析師 |
| **目標/需求** | 取得結構化的就業市場數據、追蹤長期趨勢、支持研究或政策制定 |
| **痛點** | 公開數據分散、缺乏跨國比較視角、AI 衝擊研究資料不足 |
| **使用場景** | 研究引用、政策報告撰寫、趨勢分析 |

## 3. 差異化定位

| 項目 | 內容 |
|------|------|
| **與競品的差異** | 1) 完全自動化產出，每週更新 2) 開源透明，方法論可追溯 3) 整合全球 15+ 資料來源（非單一市場） 4) 明確標註 AI 取代風險向量 |
| **獨特優勢** | 1) 免費公開使用 2) 資料來源與分析邏輯完全透明 3) 自動化確保持續更新 4) 五種 AI 取代向量分類框架 5) 跨國資料整合視角 |
| **不做什麼** | 1) 不提供個人化職涯諮詢 2) 不保證數據 100% 準確 3) 不取代專業職涯顧問 4) 不提供求職媒合服務 5) 不做付費牆或廣告變現 |

## 4. 品牌調性

| 項目 | 描述 |
|------|------|
| **語氣** | 客觀理性、數據導向、謹慎務實（避免過度樂觀或悲觀） |
| **視覺風格** | 簡約專業、資訊圖表化、儀表板風格 |
| **關鍵字** | 數據驅動、透明可追溯、AI 時代、技能轉型、客觀分析 |
