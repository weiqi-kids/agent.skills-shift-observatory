## 1. 核心資訊

| 項目 | 內容 |
|------|------|
| **Mode 名稱** | industry_segments / 產業分層分析 |
| **一行描述** | 每週橫切 14 個產業，用職缺數、薪資、AI 衝擊評級讓你知道哪個產業現在真的在招人、給多少、有多危險 |
| **核心價值主張** | 產業之間的機會落差比大多數人以為的更大，而且每週都在變動。產業分層分析是唯一同時提供台灣 14 個產業的用人方向、薪資對標與 AI 衝擊程度的週頻免費報告 |

### 價值主張細化

- **給求職者選產業**：「科技業」太籠統，半導體 vs 軟體 SaaS vs 電信的機會截然不同——用數據選產業，不靠感覺
- **給投資人 / 分析師**：就業數據是最早期的景氣信號之一，哪個產業開始大量招人，6 個月後財報數字就會跟上
- **給 HR 做薪資對標**：你的薪資 package 在本產業的 P25/P50/P75 在哪裡？和相鄰產業比呢？

## 2. 目標受眾

### 主要受眾

#### 受眾 A：正在選擇目標產業的求職者 / 轉職者

| 面向 | 說明 |
|------|------|
| **人口特徵** | 25-40 歲，有 3-10 年工作經驗，考慮跨產業轉職或進入新產業 |
| **核心需求** | 客觀依據判斷哪個產業現在最需要人、薪資最好、AI 衝擊最小 |
| **痛點** | 各產業的招募旺季、薪資水準、未來發展不透明；只能靠 LinkedIn 爬文或問朋友圈 |
| **使用場景** | 決定投遞哪個產業的職缺；跨產業轉職的目標產業評估；判斷目前所在產業是否需要離開 |
| **行動觸發點** | 看到自己目前產業的 AI 衝擊評級是「高」，同時看到另一產業的職缺量在擴張 |

#### 受眾 B：企業 HR 主管 / 薪資對標負責人

| 面向 | 說明 |
|------|------|
| **人口特徵** | HR 總監、薪酬福利主管、招募策略負責人 |
| **核心需求** | 本產業的薪資水準是否有競爭力？本產業的招募熱度與相鄰產業相比如何？ |
| **痛點** | 薪資調查報告一年一次，缺乏週頻的外部對標數據；不同產業的薪資差異難以系統性比較 |
| **使用場景** | 薪資帶調整依據、JD 薪資範圍設定、招募旺季判斷、競爭對手產業監控 |

### 次要受眾

#### 受眾 C：產業分析師 / 研究員 / 投資人

| 面向 | 說明 |
|------|------|
| **人口特徵** | 市場研究員、產業分析師、VC / PE 投資人、智庫研究員 |
| **核心需求** | 就業數據作為產業景氣的領先指標；融資信號與招募擴張的相關性分析 |
| **痛點** | 就業數據通常有 1-3 個月時效延遲，難以作為即時決策依據 |
| **使用場景** | 產業研究報告補充數據、投資標的產業動態監控、產業擴張收縮信號追蹤 |

#### 受眾 D：政府政策制定者 / 勞動部門

| 面向 | 說明 |
|------|------|
| **核心需求** | 了解哪些產業出現人才供需缺口，作為職業訓練補助方向的參考 |
| **使用場景** | 產業人才政策評估、職業訓練優先產業選定 |

## 3. 差異化定位

### 相對競爭者的差異

| 競爭者類型 | 代表 | 我們的差異 |
|------------|------|------------|
| **人力資源顧問報告** | Hays 產業薪資報告、104 職缺熱搜產業榜 | 週頻 vs 年度；14 產業系統性比較 vs 個別產業聚焦；免費 vs 付費 |
| **政府統計** | 主計處產業就業統計、勞動部產業調查 | 1-2 週時效 vs 月/季報；AI 衝擊評級整合 vs 純就業數字 |
| **媒體報導** | 科技媒體產業裁員/擴招新聞 | 系統性 14 產業全覆蓋 vs 個別事件；量化數據 vs 定性報導 |
| **職缺平台** | 104 產業職缺量、LinkedIn 產業洞察 | 跨平台多來源整合；全球對標；AI 取代向量分析 |

### 核心獨特優勢

1. **14 產業系統性比較**：唯一每週同時比較 14 個台灣主要產業的免費公開報告
2. **AI 衝擊程度評級**：每個產業都有 AI 衝擊評估（高/中/低），連結 5 種 AI 取代向量分析
3. **融資信號 + 職缺量雙軌對比**：新創融資是職缺擴張的領先信號，本報告同時呈現兩者關係
4. **全球對標層**：台灣半導體產業的用人趨勢與美國、OECD 全球數據對比，判斷是台灣特有還是全球趨勢
5. **P25/P50/P75 薪資帶**：不只說「科技業薪資高」，給出具體分位數讓薪資判斷有依據

### 我們不做的事

- 不對特定公司做個別評鑑或推薦
- 不給出「哪個產業最好」的定性結論（只給數據，讓讀者判斷）
- 不預測特定產業未來的景氣（只描述當週觀測結果）
- 不涵蓋產業定義邊界模糊的案例（如跨產業的集團型企業）

## 4. 品牌調性

### 語氣 (Tone of Voice)

| 維度 | 描述 |
|------|------|
| **整體調性** | 產業分析師的客觀嚴謹 + 商業顧問的決策導向。數據中立呈現，解讀提供多角度，但不迴避有意義的結論 |
| **對求職者說話** | 連結產業數據與個人求職決策，說明「這個數字對你的求職意味著什麼」 |
| **對 HR 說話** | 薪酬設計語境，協助做薪資對標決策 |
| **對分析師說話** | 嚴格標注數據方法論局限性，提供原始數據出處 |
| **應避免** | 對產業做道德評判；選邊站；誇大單一週數據的意義（需與歷史趨勢對比） |

### 視覺語言

| 元素 | 風格 |
|------|------|
| **產業矩陣視圖** | 14 產業 × 多維度的比較表格，讓一眼掃描成為可能 |
| **AI 衝擊熱力圖** | 以顏色深淺表示 AI 衝擊程度，直觀傳達風險層級 |
| **擴張/收縮箭頭** | 每個產業的職缺趨勢用方向箭頭快速識別 |
| **薪資分位數視覺化** | P25/P50/P75 的橫條圖，讓薪資帶對比直觀 |

### 關鍵詞

- 產業分析、產業比較、用人趨勢
- 台灣各產業職缺、產業薪資水準
- AI 衝擊評估、產業轉型、數位轉型
- 半導體、軟體業、金融業就業市場
- 週頻產業數據、產業景氣信號

## 6. 改版目標

### P0（阻擋發布的問題）

- [ ] 14 個產業必須系統性覆蓋，即使某產業本週數據不足也需說明（而非省略）
- [ ] AI 衝擊評級必須有具體依據（哪個取代向量、影響哪些角色），不能只給一個「高/中/低」結論
- [ ] 薪資數字必須附帶「面議排除聲明」，避免讀者誤解數據覆蓋率

### P1（重要優化，下次迭代完成）

- [ ] 增加產業職缺量的 12 週歷史走勢圖，讓本週數字有縱深對比
- [ ] 加入「投資信號」專欄：融資事件 + 職缺擴張的相關性分析
- [ ] 建立產業評級系統（如 A/B/C 三級），讓讀者快速掌握本週各產業的綜合狀態
- [ ] 提供「產業對比」功能：讀者可選擇 2-3 個產業做橫向比較

### P2（長期優化）

- [ ] 加入產業的季節性調整，排除農曆年、暑假等週期性因素的干擾
- [ ] 建立「產業轉型追蹤器」：長期監控特定產業的技能需求轉型（如傳統製造業的數位化進程）
- [ ] 英文版產業摘要，針對外資和全球研究者提供台灣產業數據的英文入口
