## 1. 核心資訊

| 項目 | 內容 |
|------|------|
| **Mode 名稱** | climate_index / 就業景氣溫度計 |
| **一行描述** | 每週綜合 14 個資料來源，將台灣與全球就業市場狀態濃縮為一個可行動的「溫度」判讀 |
| **核心價值主張** | 在資訊過載的時代，只需 30 秒就能掌握本週就業市場的整體走向——不是新聞聚合，而是數據驅動的溫度計 |

### 價值主張細化

- **給 HR 主管**：每週一早上看一次，馬上知道市場是在搶人還是冷靜，調整招募策略不用等季報
- **給求職者**：現在是找工作的好時機嗎？哪些產業正在擴張？一個溫度指標取代翻遍新聞的時間
- **給政策研究者**：系統性、可追蹤的週頻數據，補足官方月/季報的時效空缺

## 2. 目標受眾

### 主要受眾

#### 受眾 A：企業 HR 主管 / 招募負責人

| 面向 | 說明 |
|------|------|
| **人口特徵** | 企業 HR 總監、招募經理、HRBP，30-50 歲，管理 5-200 人的招募作業 |
| **核心需求** | 快速判斷外部市場動態，做出招募時機、薪資開價、人才搶奪策略的決定 |
| **痛點** | 季報太慢、人力資源顧問報告太貴、新聞太雜，缺乏每週更新的整合指標 |
| **使用場景** | 週一晨會前快速瀏覽，向主管報告「市場溫度」作為招募預算審核依據 |
| **行動觸發點** | 市場溫度突變（如從「溫暖」降為「偏冷」）時，重新評估職缺開放數量 |

#### 受眾 B：積極求職者 / 考慮轉職者

| 面向 | 說明 |
|------|------|
| **人口特徵** | 25-40 歲，有 3-10 年工作經驗，正在考慮轉職或主動投遞中 |
| **核心需求** | 知道現在是不是好的求職時機，哪些產業在招人，不要在市場最差時投入大量心力 |
| **痛點** | 身邊朋友都說「現在市場很差」，但不確定自己的產業是否真的受影響 |
| **使用場景** | 評估是否要現在辭職、是否要接受這個 offer、確認某產業是否真的在擴張 |
| **行動觸發點** | 目標產業出現在「擴張亮點」中，增加求職積極性 |

## 3. 差異化定位

### 相對競爭者的差異

| 競爭者類型 | 代表 | 我們的差異 |
|------------|------|------------|
| **人力資源顧問年報** | Hays、Manpower、104人力銀行年度報告 | 週頻更新 vs 年度/季度；免費 vs 付費；具體溫度判讀 vs 趨勢描述 |
| **政府統計** | 主計處、勞動部月報 | 1-2 週時效 vs 1-3 個月時效；多來源綜合 vs 單一官方數據 |
| **財經媒體** | 經濟日報、工商時報就業新聞 | 系統性整合 vs 個別事件報導；量化溫度 vs 質性描述 |
| **求職平台統計** | 104職缺熱搜、1111薪資指南 | 跨平台多來源 vs 單平台數據；全球對標 vs 台灣單一市場 |

### 核心獨特優勢

1. **「溫度計」概念**：將複雜的多指標市場狀態濃縮為 5 級溫度，任何人 30 秒可理解
2. **台灣 + 全球雙軌對比**：同一份報告同時看台灣與全球趨勢，判斷是台灣特有問題還是全球性趨勢
3. **AI 取代向量交叉分析**：不只說「市場偏冷」，還說「認知例行類職缺比認知非例行類職缺更冷」
4. **每週自動更新**：不依賴人工整理，14 個資料源自動聚合
5. **方法論透明**：溫度判讀依據公開，資料來源具名，讀者可自行驗證

### 我們不做的事

- 不預測未來特定時間點的就業數字（避免誤導）
- 不推薦特定招募管道或人力仲介
- 不做產業或公司評等（個別公司評鑑）
- 不提供針對特定個人情況的建議

## 4. 品牌調性

### 語氣 (Tone of Voice)

| 維度 | 描述 |
|------|------|
| **整體調性** | 專業分析師的判斷力 + 天氣預報的親切感。數據嚴謹但不學術；有立場但不武斷 |
| **對 HR 說話** | 同儕對話，假設對方懂市場術語，直接給策略含意 |
| **對求職者說話** | 清楚解釋，避免行話，強調「這對你意味著什麼」 |
| **對研究者說話** | 透明方法論，明確標示數據局限性 |
| **應避免** | 過度樂觀的市場炒作語氣；聳動標題（「職缺大崩盤！」）；過度技術性的統計術語 |

### 視覺語言

| 元素 | 風格 |
|------|------|
| **溫度符號** | 5 色表情（🔴🟠🟡🟢🔵）為核心視覺錨點，一致且直觀 |
| **數據表格** | 清晰結構化，對比欄位突出變化方向 |
| **色彩使用** | 溫暖色（擴張）/ 冷色（收縮），直覺對應 |
| **密度** | 資訊豐富但分層：摘要 → 核心指標 → 深度分析，讓不同深度的讀者都能找到出口 |

### 關鍵詞

- 就業市場溫度、景氣指數、就業景氣、市場動態
- 台灣職缺趨勢、全球就業市場
- AI 取代風險、產業擴張、裁員信號
- 每週更新、多來源整合、數據驅動
