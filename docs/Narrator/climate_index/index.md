# 景氣溫度計

## 這份報告是什麼？

景氣溫度計綜合多來源數據，用「溫度」來形容就業市場的整體狀態——讓你快速掌握「現在市場好不好」。

## 適合誰看？

- ✅ 正在找工作的**求職者**
- ✅ 考慮跳槽的**在職者**
- ✅ 關注市場動態的 **HR**
- ✅ 研究就業趨勢的**研究者**

## 你會看到什麼？

| 內容 | 說明 |
|:-----|:-----|
| **本週溫度** | 寒冷 🔴 → 偏冷 🟠 → 持平 🟡 → 溫暖 🟢 → 過熱 🔵 |
| **核心指標** | 職缺數量、薪資變化、失業率等 |
| **產業信號** | 哪些產業在擴張、哪些在收縮 |
| **重大事件** | 本週影響市場的裁員/招聘/融資事件 |
| **AI 取代向量** | 各類型工作的風險變化 |
