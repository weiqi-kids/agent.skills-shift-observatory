# 薪資帶

## 這份報告是什麼？

薪資帶分析各職業的薪資分布（P25 / P50 / P75）——讓你了解「同樣職位的薪資範圍大概多少」。

## 適合誰看？

- ✅ 薪資談判前的**求職者**
- ✅ 做薪酬對標的 **HR**
- ✅ 協助客戶的**薪酬顧問**
- ✅ 調整薪資結構的**企業主管**

## 你會看到什麼？

| 內容 | 說明 |
|:-----|:-----|
| **53 角色薪資帶** | 各職業的 P25 / P50 / P75 |
| **跨產業比較** | 同職位在不同產業的薪資差異 |
| **跨地區比較** | 台北 vs 新竹 vs 其他地區 |
| **薪資成長趨勢** | 上升/下降最多的職業 |
| **全球對標** | 台灣 vs 美國 vs 全球 |

## 術語說明

| 術語 | 意義 |
|:-----|:-----|
| **P25** | 25% 的人低於這個薪資 |
| **P50** | 中位數，一半高於、一半低於 |
| **P75** | 75% 的人低於這個薪資 |
