## 一、競品總覽

本分析識別了 Skills Shift Observatory 在不同報告面向的主要競爭對手，分為三類：
1. **全球勞動力情報平台**：Lightcast、LinkedIn Economic Graph、JobsPikr
2. **薪資與招聘顧問報告**：Robert Half Salary Guide、Michael Page Salary Guide、Hays Salary Guide
3. **台灣本地就業資訊服務**：104 職場力、1111 調薪調查、經濟日報就業報導

## 二、競品個別分析

### 競品 A：LinkedIn Economic Graph
**URL**：https://economicgraph.linkedin.com/workforce-data

| 項目 | 描述 |
|------|------|
| **定位** | LinkedIn 職業社群資料驅動的全球勞動力情報平台 |
| **目標受眾** | 政策制定者、企業 HR、學術研究者、經濟分析師 |
| **更新頻率** | 月更（Workforce Report）；部分指數雙週更（Workforce Confidence Index） |
| **資料深度** | 超過 2 億美國會員的職業履歷 + 全球職缺數據；覆蓋美國、英國、印度、EMEA、APAC |
| **獨特功能** | Career Explorer 互動工具、技能需求追蹤、AI 勞動力季度追蹤 |
| **弱點** | 以英語市場為主；台灣深度不足；資料主要來自 LinkedIn 用戶（偏向科技與白領）；非中文介面 |
| **授權模式** | 部分免費（公開報告）；深度數據需 LinkedIn Talent Insights 付費訂閱 |

**與本站競爭面向**：skills_drift（技能需求）、industry_segments（勞動力趨勢）

## 四、競爭優勢分析

### 4.1 本站差異化優勢（Unique Value Propositions）

1. **週度更新 + 結構化框架**：市場上唯一以週為單位、系統化追蹤技能、薪資、產業、市場溫度的中文服務
2. **AI 取代向量框架**：獨創五向量分析（認知例行、認知非例行、體力例行、體力非例行、人際互動），提供結構化的 AI 衝擊評估
3. **台灣 + 全球雙軌觀測**：兼顧本地市場（tw_govjobs）與全球招聘信號（HN Hiring、Arbeitnow 等），具備跨市場比較能力
4. **方法論透明**：資料來源、層數、信心程度均公開，Qdrant 搜尋過程可追溯
5. **免費開放**：無付費牆，GitHub Pages 架設，長期維護成本低

### 4.2 本站相對弱點

1. **台灣深度不足**：tw_104_jobs、tw_company_reviews 資料源受限，台灣專業人才市場觀測薄弱
2. **資料規模較小**：週度樣本數（~5,000 筆職缺）遠低於 Lightcast（25 億）或 LinkedIn（2 億會員）
3. **無互動功能**：缺乏薪資計算機、職業探索工具等互動元素
4. **品牌知名度低**：新興服務，尚無媒體報導或機構引用
5. **無訂閱/通知機制**：讀者無法設定週報提醒或自訂追蹤職業

## 六、市場空白（機會）

1. **台灣中文就業情報週報空白**：競品中無一提供系統化、週度更新的中文就業情報；最接近的是 104 職場力，但缺乏統一分析框架
2. **AI 衝擊結構化追蹤空白**：全球競品多在研究報告中提及 AI 衝擊，但無人以週度頻率系統追蹤分職業、分向量的影響
3. **台灣薪資週度更新空白**：台灣薪資資料主要來自年度報告（Michael Page、104 薪資白皮書），週度薪資變化追蹤為本站獨有
4. **免費開放的産業分層追蹤**：Lightcast 等競品的產業職缺分析需付費，本站免費提供週度產業快照
