## 4. 頁面規劃

### 4.1 週報「本週行動清單」區塊

| 項目 | 內容 |
|------|------|
| 頁面 URL | 所有 /reports/*-w{XX}/ |
| 類型 | 修改 |
| 所屬階段 | Phase 1 |
| 優先級 | P0 |

#### 頁面策略

| 項目 | 說明 |
|------|------|
| 區塊目標 | 將報告洞察轉化為可執行的行動建議 |
| 目標受眾 | 求職者、在職者 |
| 關鍵訊息 | 1. 本週最重要的 1 件事 / 2. 3-5 條具體行動 / 3. 下週關注點 |
| 位置 | 報告末尾，總結之前 |

#### 內容規格

```markdown
## 本週行動清單

### 求職者
- [ ] {具體行動 1}
- [ ] {具體行動 2}
- [ ] {具體行動 3}

### 在職者
- [ ] {具體行動 1}
- [ ] {具體行動 2}

### 下週關注
- {預告下週重點}
```
